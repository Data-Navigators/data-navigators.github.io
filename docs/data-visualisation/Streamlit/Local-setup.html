Streamlit Local Setup

There are two ways to setup and run the code in this repository:

  1. Setup Directly on the Machine: Install all the Software directly on your machine and run the code.
  2. Setup Using Docker: Use a pre-built Docker image containing all the Software already and run the code inside Docker container.

Setup Directly on the Machine‚Äã

Prerequisites‚Äã

  • Pyenv
  • Python
  • Poetry

Ensure you have Git installed on your machine. If not, you can download it from git-scm.com.

Ensure you have Git installed on your machine. If not, you can download it from git-scm.com.

Open your terminal.

Navigate to the directory where you want to clone the project. For example:

cd path/to/your/directory

Clone the repository using this command:

git clone https://github.com/Data-Dynamos/data_dynamos_exercise.git

Create a new project and set the root folder of project as working directory‚Äã

cd data_dynamos_exercise

Install Pyenv v2.3.22‚Äã

curl https://pyenv.run | PYENV_GIT_TAG=v2.3.22 bash
Load pyenv automatically‚Äã
For bash‚Äã
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(pyenv init -)"' >> ~/.bashrc
exec "$SHELL"
For Zsh‚Äã
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(pyenv init -)"' >> ~/.zshrc
exec "$SHELL"

If you are using shells other than bash or Zsh refer: https://github.com/pyenv/pyenv#set-up-your-shell-environment-for-pyenv

Install Python 3.8.16‚Äã

pyenv --version
pyenv install 3.8.16
pyenv versions # should list Python 3.8.16 version

Install Poetry 1.5.1‚Äã

curl -sSL https://install.python-poetry.org | POETRY_VERSION=1.5.1 python3 -
Add Poetry bin directory to PATH‚Äã
For bash:‚Äã
echo 'export PATH="$HOME/.local/bin/:$PATH"' >> ~/.bashrc
exec "$SHELL"
For Zsh:‚Äã
echo 'export PATH="$HOME/.local/bin/:$PATH"' >> ~/.zshrc
exec "$SHELL"
